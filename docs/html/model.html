class pyasv.model.deep_speaker.DeepSpeaker(config, out_channel=[64, 128, 256, 512])[source]¶
__init__(config, out_channel=[64, 128, 256, 512])[source]¶

Create deep speaker model.

config : config class.
The config of ctdnn model, no extra need.
out_channel : list

The out channel of each res_block.

default

out_channel = [64, 128, 256, 512]

Note

There is no predict operation in deep speaker model. Because we use the output of last layer as speaker vector. we can use DeepSpeaker.feature to get these vector.
