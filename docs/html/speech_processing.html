Speech Processing¶

speech_processing.ext_mfcc_feature(config)¶

This function is used for extract MFCC feature of a dataset.

Parameters:
  • url_path – The path of the ‘PATH’ file. The file contain the path to all audio and its number in the dataset. The contents of the file should be as follows: xxxxx/your_data_path/1_1.wav 0 xxxxx/your_data_path/1_2.wav 0 xxxxx/your_data_path/2_1.wav 1
  • config – The type is config class. We only need to know the save path, so we can use same config with model.
Return mfccs:

The mfcc feature of this dataset. Each frame concat with its before and after frames.
Return labels:

The labels of this dataset.
