Data manager¶

class pyasv.data_manage.DataManage(raw_frames, raw_labels, config)[source]¶

Use DataManage to manage normal dataset, we use next_batch to get batch data every step.

__init__(raw_frames, raw_labels, config)[source]¶
raw_frames : list or np.ndarray
the feature array of a dataset.
raw_labels : list or np.ndarray
the label array of a dataset.
config : config class
The config of your model, we only need use its ‘batch_size’ member.
next_batch¶

property to get next batch data.

batch_frames: list or np.ndarray. batch_labels: list or np.ndarray.

Note

the type of batch_frames and batch_labels is same as your input data.

class pyasv.data_manage.DataManage4BigData(config)[source]¶

Use DataManage4BigData to manage huge dataset. we will save the dataset to disk, and restore a batch of them in each step we can still use next_batch to get batch data every step.

__init__(config)[source]¶
config: config class
the config of your model. we will use its batch_size to manage our data and save the data to save_path/data.
next_batch¶

Read a batch of data.

frames : np.ndarray
the feature array of your dataset.
labels : np.ndarray
the label array of your dataset.

Note

we will check if the data is saved. If you didn’t save the data you will get log and two empty array.

write_file(raw_frames, raw_labels)[source]¶

Save your data to save_path/data.

raw_frames : list or np.ndarray
the feature array of your dataset.
raw_labels : list or np.ndarray
the label array of your dataset.
