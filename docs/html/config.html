Config¶

The config class is used for set our model and operation. In general, we only need a part of his parameters. So, we just need to set the needed parameters when we create a config class.

Param:

config_path: The path of config which you want to restore, If you want to create a new config, this param should be None.
Parameters:
  • is_big_dataset – If we store the features to disk and restore a part of feature each step. The type is bool
  • url_of_bigdataset_temp_file – To decide the save path of the features you want to restore. If the param “is_big_dataset” is False, this param should be None.
